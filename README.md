# Exam-Prep-Platform

[![Quality Gate Status](http://localhost:9000/api/project_badges/measure?project=PTE-Exam-Prep-Platform&metric=alert_status&token=sqb_9efd72a1afef5e85ffaaaf3496ddd63595617958)](http://localhost:9000/dashboard?id=PTE-Exam-Prep-Platform)

An interactive platform for PTE Exam preparation, featuring practice sections, mock tests, and a student forum.

## Running the Project

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Run tests:
   ```bash
   npm test
   ```
